- Yozaru voicebanks 🎤 -

Yozaru (from ヨザル meaning Night monkey) is an Utau voicebank created by Yoazur avaliable in multiple languages.
It has 3 voice colors : Natural, Soft and Solid for all voicebanks. Is is highly recommended to use these voicebanks using OpenUtau for ease of use.

Available voicebanks:

  • Yozaru Japanese

    Recorded using Salem's VCV and CVVC reclists.

  • Yozaru French
  • Yozaru English (still a work in progress...)
